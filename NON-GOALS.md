# Non-Goals

CICULLIS explicitly does NOT aim to:

## Functionality non-goals

- Replace human review
- Provide security scanning
- Offer code quality metrics
- Generate reports or dashboards
- Integrate with issue trackers
- Send notifications
- Store historical data
- Provide remediation suggestions

## Architecture non-goals

- Plugin system
- Configuration UI
- Database backend
- API endpoints
- Webhook integrations
- Multi-tenancy
- User authentication

## Scope non-goals

- General-purpose CI tool
- Test framework
- Linting system
- Dependency scanner
- License checker
- Performance analyzer

## Evolution non-goals

- Feature roadmap
- Backward compatibility promises beyond current version
- Migration tooling
- Deprecation warnings
- Gradual rollout mechanisms

## CICULLIS is a gate, not a platform.
